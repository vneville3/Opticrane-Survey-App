How this app works

  1. Add to Home Screen (iPhone): Open in Safari → Share → Add to Home Screen.
  2. Fill on site: Contact, Site Location, Type of Machine, Issues, Urgency, Reason.
  3. Assessment: Score A/B/C appears automatically with a suggested kit (editable).
  4. Queue Submission: Saves locally (works offline).
  5. Sync: Set Endpoint + optional token → Sync Now.
  6. Backfill: Opens your Rep Backfill form link for extra notes/photos.
  7. Where files live: Drafts/queue are local; use Export CSV/JSON for backup.
